# IBM Quantum Developer Practice Exam

Interactive multiple-choice practice website for the **IBM Certified Quantum Computation using Qiskit v2.X Developer – Associate** certification.

## Features

- **3 Exam Sections** · 60 questions total
- **Instant Scoring** with pass/fail feedback (70% threshold)
- **Detailed Explanations** for every answer
- **MathJax** rendering for quantum notation ($|0\rangle$, matrices, etc.)
- **Progress Tracker** bar while answering
- No build tools required — pure HTML/CSS/JS

## Sections

| Section | Topic | Questions |
|---------|-------|-----------|
| 1 | Pauli Operators & Quantum Operations | 20 |
| 2 | Pauli Operators & Quantum Operations (Practice Set 2) | 20 |
| 3 | Advanced Circuit Construction & Transpilation | 20 |

## Live Site

Hosted via GitHub Pages. Access at: `https://<owner>.github.io/<repo>/`

## Local Development

No build step required. Open `index.html` directly in a browser, or serve with any static file server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
